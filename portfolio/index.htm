NJTPA Geoportal

 The public face of the Enterprise-Class Geospatial Database Development (EGIS) Project for the North Jersey Transportation Planning Authority. This applications allows users to search metadata by keyword, category or location. It also provides functionality for clipping selected data layers in an area of interest and sending the clipped layers to an email address as well as a full inventory of the EGIS data catalog.

Sample Airline Travel Site

 The final project for the NYU .NET Application Development certificate program. The assignment was for an ASP.NET/SQL Server application that would display flight information using a calendar control, allow a user to select flights to purchase, generate a confirmation number and have user logins.

Fantasy Football Auction Draft Application

 An ASP.NET application for managing a live auction fantasy football draft. This sample application utilizes ASP.NET web forms, a SQL Server backend with custom views and stored procedures, jQuery Effects. There is also a mobile version of the application built with the iUi framework that is automatically directed when viewed from a mobile device
